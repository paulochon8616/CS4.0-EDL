  • 10: mixing length model
  • 20: standard model
  • 21: model with Linear Production (LP) correction
  • 30: (LRR)
  • 31: (SSG)
  • 32: (EBRSM)
  • 40: LES (constant Smagorinsky model)

  • 42: LES (WALE)

  • 51: v2f
  • 60: SST
  • 70: Spalart-Allmaras model
